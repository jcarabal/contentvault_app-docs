Before Contacting Support

To help us assist you faster, please:

  • Check the FAQ section first
  • Review the Configuration Guide
  • Note your app version (Settings ‚Üí About)
  • Document any error messages you're seeing
  • Describe the steps to reproduce the issue

Report a Bug

Found a bug? We want to know! Please include:

  • Device: iOS/Android/macOS/Web
  • Version: Your app version number
  • Description: What happened vs. what you expected
  • Steps: How to reproduce the issue
  • Screenshots: If applicable

Email bug reports to: bugs@vaulttrak.com
